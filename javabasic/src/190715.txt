## 수업 개요
- 비주얼라이제이션

### 기술을 중심으로 한 이 교육과정의 구조
- DB에서 오라클을 통해 웹서버로 정보 공유 -> 자바로 구현
- 웹클라이언트 pc -> HTML5, CSS3, Javascript, JSON, jQuery로 구현
- 웹클라이언트 모바일 -> 반응형 웹 디자인으로 구현
- 웹클라이언트 <-> 웹서버 JSP로 구현

#### MVC 기반의 웹 만들기
- M 모델 : DOM
- V 보이는 것 : 뷰
- C 정보 가공 : 컨트롤
- 스프링 : 전자정보프레임워크를 만듦.

#### 웹 만들기
- 위지위그 라는 툴로 쉽게 만들 수 있음
- 하지만 구조를 이해하고 소스를 편집할 수 있게 하기 위해 HTML 공부


## HTML이란
- Hyper Text Markup Language
- 마크업 : 태그를 사용해 문서에서 어느 부분이 제목이고 본문인지, 어느 부분이 사진이고 링크인지 표시하는 것

#### 버전의 표시
- 버전이 n.m.c로 표시된다고 했을 때, 숫자의 순서는 중요도 순서
- 버전 1.0.0 -> 1.0.1 : 변경사항 생겼지만 중요도 낮음
- 버전 1.0.0 -> 1.1.0 : 변경사항 생겼으며 중요도 중간
- 버전 1.0.0 -> 2.0.0 : 변경사항 생겼으며 중요도 높음
